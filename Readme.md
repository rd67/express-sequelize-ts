# Usage (run fullstack app on your machine)

## Prerequirements in server/package.json

## Server-side Usage(PORT: 8000)

### Starting Server and Making Build

```terminal
$ cd server                 // go to server folder
$ npm i                     // npm install pacakges
$ cp .env.example .env      // Update .env file keys
$ npm run dev               // run it locally

// deployment for server app
$ npm run build // this will build the server code to es5 js codes and generate a build folder in server folder
```
